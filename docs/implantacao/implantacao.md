# Implantação
* A implantação do software será feita em multiplataforma, com a intenção de facilidar o acesso mobile, estará disponivel em todas as versões IOS, ANDROD E WINDOWS PHONE.
através dos aplicativos de busca de cada sistema operacional.

* Iniciaremos em fase de teste ou mais conhecido como fase "beta", onde o software estará disponivel para pequenas modificações tornando-o mais dinâmico, dentro do aplicativos
será disponiveis diversos canais de sugestões para que o software se adapte as necessidades da maioria de nosso clientes.
